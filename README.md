# YouTube Ad Skipper Pro: TUFF Edition V3.5

![Ad Skipper Shield](https://img.shields.io/badge/Ad%20Skipper-TUFF%20V3.5-d4af37?style=for-the-badge) ![Manifest V3](https://img.shields.io/badge/Manifest-V3-1e1e1e?style=for-the-badge)

Una extensión de navegador premium, agresiva y de altísimo rendimiento para evadir, ocultar y saltar anuncios automáticamente en YouTube de manera invisible.

## 🔥 Características Principales
- **Motor de Salto Invisible (Zero-Delay):** Destruye automáticamente los anuncios interrumpiendo el reproductor sin demoras y simulando clics reales para evadir bloqueos.
- **Ocultación Dinámica y Extrema:** Destruye `Sponsored Shorts`, banners laterales y overlays nativos a través de CSS puro e inyectado.
- **Tema Reactivo en Tiempo Real:** Elige entre Dorado Pulido, Plateado Metálico, Negro Carbono o Rojo Clásico. Toda la barra de reproducción de YouTube (e incluso los Shorts) cambiará instantáneamente de color en coordinación con la ventana del popup.
- **Notificación Flotante Estética:** Recibe un badge interactivo y fluido directamente en la pantalla de video cuando un anuncio es destruido.
- **Persistencia Tuff:** Estadísticas de saltos y configuración de motor/tema guardadas en caché local.
- **Identidad Visual TUFF:** Una Interfaz de Usuario industrial, color grafito/negro mate, compacta y profesional.

## 🛠 Instalación Local (Modo Desarrollador)
1. Ve a la página de extensiones en tu navegador Chrome: `chrome://extensions/`.
2. Activa el **Modo de desarrollador** en la esquina superior derecha.
3. Selecciona **Cargar descomprimida** (Load unpacked).
4. Selecciona la carpeta raíz de este repositorio (`youtube-adblocker`).
