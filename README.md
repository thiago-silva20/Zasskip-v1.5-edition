A high-performance Google Chrome extension built entirely on the modern **Manifest V3** architecture. This project completely optimizes your video streaming experience by eliminating interruptions and adding an advanced, real-time aesthetic customization layer.

---

## ✨ Key Features

* **⚡ Advanced Ad Skipping:** Seamlessly detects and skips traditional video advertisements in milliseconds, completely unseen.
* **✂️ Shorts Cleanup:** Automated removal of sponsored Shorts and promotional feed content using a reactive system powered by `MutationObserver`.
* **🎨 Real-Time Theme Selector:** An integrated dashboard (Popup UI) featuring 4 premium color variants that dynamically change the video player bar and scrubber on the fly (Gold, Silver, Carbon Gray, and Classic Red).

---

## 🛠️ Technologies & Architecture

* **Extension Architecture:** Google Chrome Manifest V3 (Ensuring future-proof performance and security compliance).
* **Core JavaScript (Asynchronous):** Advanced implementation of `chrome.storage.local` for theme persistence, global `chrome.storage.onChanged` listeners, direct DOM manipulation, and seamless stylesheet injection.
* **UI/UX Design:** HTML5 and CSS3 structured with **Custom Properties (CSS Variables)** to allow instantaneous, fluid color-swapping directly from the extension's popup window.

---

## 🚀 How to Install (Developer Mode)

Since this is a custom portfolio project, you can easily load it locally on your machine:

1. **Download** this repository as a `.zip` file and extract it on your computer.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. In the top-right corner, toggle the **"Developer mode"** switch to ON.
4. Click the **"Load unpacked"** button in the top-left corner.
5. Select the root folder where you extracted the project files (the folder containing `manifest.json`).
6. Open YouTube, click the **Zasskip** icon, and start customizing your player!
